  <h3 class="book-title text-[var(--book-title-1)] text-3xl mb-4 mt-6">Solución del Problema 6</h3>

  <p>Las conclusiones clave de este ejercicio son:</p>

  <ul>
    <li><strong>El nivel freático</strong> en un acuífero libre es una réplica atenuada de la topografía, inclinándose hacia el punto de descarga (el río en este caso).</li>
    <li><strong>Los contornos equipotenciales</strong> son líneas de igual carga hidráulica. En el ejercicio interactivo, se dibujan seleccionando pares de puntos a la misma elevación.</li>
    <li><strong>Las líneas de flujo</strong> son perpendiculares a las líneas equipotenciales e indican la dirección del flujo de agua subterránea — desde una carga hidráulica mayor hacia una menor.</li>
    <li><strong>En un acuífero libre cercano a un río</strong>, el agua subterránea fluye hacia el río, que actúa como frontera de descarga. Las líneas equipotenciales se curvan al acercarse al río, reflejando la geometría del sistema de flujo.</li>
    <li><strong>La carga hidráulica</strong> disminuye a lo largo de una línea de flujo desde el área de recarga hacia el punto de descarga. El ejercicio interactivo permite medir esta diferencia (Δh) entre puntos seleccionados.</li>
  </ul>

  <h4 class="book-title text-[var(--book-title-1)] text-xl mt-6 mb-2">Conexiones Conceptuales</h4>
  <ul>
    <li>Compara este ejercicio con el <strong>Ejercicio 5</strong>, que discute la representación del nivel freático como una superficie planar con flujo predominantemente horizontal. El Ejercicio 6 proporciona un ejemplo concreto donde el nivel freático interactúa con ríos.</li>
    <li>Compara con el <strong>Ejercicio 3</strong>, que introduce las líneas equipotenciales y la dirección del flujo en una configuración de columna 1D más simple. El Ejercicio 6 extiende estos conceptos a una sección transversal 2D con condiciones de frontera.</li>
    <li>La relación entre líneas equipotenciales y líneas de flujo demostrada aquí es la base para comprender las <strong>redes de flujo</strong> — una técnica gráfica utilizada para resolver problemas de flujo de agua subterránea (ver Poeter y Hsieh, 2020).</li>
  </ul>
